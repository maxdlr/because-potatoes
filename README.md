# Hackathon ESGI
> "École Supérieure de Génie Informatique"

## Start the project
### Install packages
```shell
composer install
```
### Launch server
```shell
composer ss
```
### Launch tests
```shell
composer tests
```
### Create ``/.env.local``
Include this.
```dotenv
DB_USER="username"
DB_PASSWORD="password"
DB_NAME="dbname"
DB_HOST="dbhost"
DB_PORT=dbport
```
